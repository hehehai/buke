# @hehehai/buke

Pake-like Electrobun CLI for packaging websites into lightweight desktop apps.

## Install

`@hehehai/buke` published on npm still runs on Bun. For a no-Bun install, use the GitHub Releases binary, Homebrew, or the shell installer from the main repository.

```bash
npm install -g @hehehai/buke
buke --help
```
